# Phishing Attack Simulation and Detection System

## Objective
To simulate phishing emails and detect malicious emails using keyword analysis and URL detection.

## Tools Used
- Python
- Regular Expressions

## Features
- Detects phishing keywords
- Identifies suspicious URLs
- Classifies emails as Phishing or Legitimate

## Output
The system successfully flags phishing emails based on predefined rules.
